# -GitHub-User-Finder

A simple Python project that uses the GitHub API to fetch user information.

## Features

- Search any GitHub user by username
- Display user's:
  - Name
  - Followers
  - Following
  - Public Repositories
  - Location
  - Company

## Technologies Used

- Python
- Requests Library
- GitHub REST API

## Installation

1. Clone the repository

```bash
git clone https://github.com/STTRLORD/GitHub-User-Finder.git
```

2. Install dependencies

```bash
pip install requests
```

3. Run the program

```bash
python send_request.py
```

## Example

Input:

```text
Enter your username -> torvalds
```

Output:

```text
Name: Linus Torvalds
Followers: 250000+
Following: 0
Public Repositories: 8
```
